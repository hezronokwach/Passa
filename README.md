### Quick Start

1. **Clone the repository:**
```bash
git clone https://github.com/yourusername/connectsphere.git
cd connectsphere
```

2. **Install frontend dependencies:**
```bash
cd frontend
npm install
```

3. **Start the development server:**
```bash
npm run dev
```

4. **Open your browser:**

```bash
npm run preview
```
